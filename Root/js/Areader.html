JavaScript - من الصفر إلى الاحتراف

مقدمة في JavaScript

JavaScript هي لغة برمجة نصية تُستخدم لجعل صفحات الويب تفاعلية. تُعد واحدة من اللغات الأساسية في تطوير الويب الحديث.

متغيرات JavaScript

في JavaScript، تُستخدم الكلمات المفتاحية var و let و const ل объявл المتغيرات:

// объявл متغير باستخدام var
var name = "أحمد";

// объявл متغير باستخدام let
let age = 25;

// объявл ثابت باستخدام const
const PI = 3.14;

دوال JavaScript

الدوال تُستخدم لتعبئة مجموعة من التعليمات البرمجية التي يمكن استدعاؤها عدة مرات:

function welcome(name) {
    return "مرحباً، " + name + "!";
}

console.log(welcome("أحمد")); // مرحباً، أحمد!

مصفوفات JavaScript

المصفوفة هي كائن يُستخدم لتخزين بيانات متعددة في متغير واحد:

let fruits = ["تفاح", "موز", "برتقال"];

console.log(fruits[0]); // تفاح
console.log(fruits.length); // 3
ملاحظة: JavaScript حساسة لحالة الأحرف، مما يعني أن myVariable و myvariable هما متغيران مختلفان.

البرمجة الكائنية (OOP) في JavaScript

البرمجة الكائنية في JavaScript تُستخدم لإنشاء كائنات وتنظيم الكود بطريقة منظمة:

class Car {
    constructor(brand, model) {
        this.brand = brand;
        this.model = model;
    }

    start() {
        console.log(this.brand + " " + this.model + " قيد التشغيل");
    }
}

let myCar = new Car("Toyota", "Camry");
myCar.start(); // Toyota Camry قيد التشغيل
تحذير: تأكد من التحقق من وجود العناصر قبل التفاعل معها لتجنب الأخطاء البرمجية.

الاستنتاج

JavaScript لغة قوية ومرنة تُستخدم في تطوير الواجهات الأمامية والخوادم. مع الاستمرار في التعلم والممارسة، يمكنك إتقان هذه اللغة وبناء تطبيقات مذهلة.

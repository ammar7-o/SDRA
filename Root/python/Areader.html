Python - من الصفر إلى الاحتراف

مقدمة في Python

Python هي لغة برمجة عالية المستوى تتميز بالyntax بسيط وسهل القراءة. تُستخدم في مجالات متعددة مثل تطوير الويب، الذكاء الاصطناعي، تحليل البيانات، وغيرها.

متغيرات Python

في Python، لا حاجة ل объявл المتغيرات باستخدام كلمات مفتاحية خاصة:

# объявл متغير
name = "أحمد"
age = 25
PI = 3.14

print(name)  # أحمد
print(age)   # 25

دوال Python

تُستخدم الكلمة المفتاحية def لتعريف الدوال في Python:

def welcome(name):
    return f"مرحباً، {name}!"

print(welcome("أحمد"))  # مرحباً، أحمد!

قوائم Python

القوائم في Python تُستخدم لتخزين بيانات متعددة في متغير واحد:

fruits = ["تفاح", "موز", "برتقال"]

print(fruits[0])       # تفاح
print(len(fruits))     # 3
ملاحظة: Python تعتمد على المسافات البادئة (indentation) لتحديد كتل التعليمات البرمجية.

البرمجة الكائنية (OOP) في Python

البرمجة الكائنية في Python تُستخدم لإنشاء كائنات وتنظيم الكود بطريقة منظمة:

class Car:
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model
    
    def start(self):
        print(f"{self.brand} {self.model} قيد التشغيل")

my_car = Car("Toyota", "Camry")
my_car.start()  # Toyota Camry قيد التشغيل
تحذير: تأكد من فهمك للمسافات البادئة في Python لتجنب الأخطاء البرمجية.

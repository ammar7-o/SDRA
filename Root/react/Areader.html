React - من الصفر إلى الاحتراف

مقدمة في React

React هو مكتبة JavaScript مفتوحة المصدر تُستخدم لبناء واجهات المستخدم (UI). تم تطويره من قبل Facebook ويُستخدم على نطاق واسع في بناء تطبيقات الويب التفاعلية.

مكون بسيط في React

في React، يتم إنشاء المكونات باستخدام دوال أو فئات:

import React from 'react';

function Welcome(props) {
  return <h1>مرحباً، {props.name}</h1>;
}

export default Welcome;

حالة المكون (State)

تُستخدم الحالة لتخزين المعلومات التي يمكن أن تتغير داخل المكون:

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>العداد: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        ارفع العداد
      </button>
    </div>
  );
}

ال Life Cycle Hooks

تُستخدم hooks لتنفيذ المهام عند نقاط مختلفة في دورة حياة المكون:

import React, { useState, useEffect } from 'react';

function MyComponent() {
  const [data, setData] = useState([]);

  useEffect(() => {
    // تُنفذ بعد التصيير
    fetchData().then(setData);
  }, []);

  return <div>...</div>;
}
ملاحظة: React يعتمد على Virtual DOM لتحسين الأداء.

نظام إدارة الحالة (Redux)

Redux هو أداة إدارة حالة مصممة لتطبيقات JavaScript:

// action
const increment = () => ({ type: 'INCREMENT' });

// reducer
const counterReducer = (state = 0, action) => {
  switch (action.type) {
    case 'INCREMENT':
      return state + 1;
    default:
      return state;
  }
};
تحذير: لا تستخدم Redux لكل مشروع، فقط عند الحاجة لإدارة حالة معقدة.
